My Cyber Secuity Interests

I developed a passion for cyber security after some medical problems which left me unable to continue my previous career. I started of by building PC's and computational machines before delving into codeing.

Blockchain Technology

One of my main interests is blockchain technology. Carrying out smart contract security audits and conducting blockchain forensics while keeping a record of malicious code and assosiated hash signature. Decompiling malicious code to understand how it interacts will a contract when a function is called. I aslo deploy contracts on a testnet to manually interact and manipulate functions to test for possible vulnerabilities.

Cyber Security Psychology

I have found that the main threat to any code is the person capable of calling functions that can alter the state of the system. Understanding peoples motives for having code that on its own is not malicious unless it is purposely changed to become malicious. Knowing the intensions of people with access to the high level functions is vital.
